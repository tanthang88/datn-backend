> ### Khách hàng tiềm năng (Guest)
>
> - Xem trang chủ
>
> - Xem các sản phẩm theo:
>>
>> - Theo danh mục
>>
>> - Nhà sản xuất
>>
>> - Tìm kiếm theo văn bản gõ vào
>>
> - Xem chi tiết sản phẩm được chọn
>
> - Xem khuyến mãi
>
> - Xem so sánh
>
> - Mua hàng
>
> - Quản lý giỏ hàng
>
> - Đăng ký tài khoản để trở thành khách hàng
>

>
> - Xem đơn hàng
>
> - Thanh toán
